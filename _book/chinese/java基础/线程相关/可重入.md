# 可重入

可重入函数保证了在多线程条件下，函数的状态不会出现错误。

以下分别是一个不可重入和可重入函数的示例：
```c
static int tmp;
void func1(int* x, int* y)
 {
  tmp=*x;
  *x=*y;
  *y=tmp;
}
```

```c
void func2(int* x, int* y)
{
  int tmp;
  tmp=*x;
  *x=*y;
  *y=tmp;
}
```

func1是不可重入的，func2是可重入的。因为在多线程条件下，操作系统会在func1还没有执行完的情况下，切换到另一个线程中，那个线程可能再次调用func1，这样状态就错了。
