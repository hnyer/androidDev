1. 虚拟机对比

1.1. JVM

Java Virtual Machine。基于栈 ，运行java字节码。

1.2. DVM

Dalvik Virtual Machine 。
运行自谷歌定义的 .dex字节码格式。

Dalvik 基于寄存器的架构， 更适合资源紧张的系统。

1.3. ART

Android runtime 。
Android 4.4 开始加入 ART模式。

ART下，应用在第一次安装的时候，字节码就会预编译成机器码，首次启动会变慢，
但是以后每次启动执行的时候，都可以直接运行。（空间换时间）
